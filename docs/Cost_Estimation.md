## 9.2 Basic COCOMO Formulas (Organic Mode)

```
Effort (E) = a × (KLOC)^b
= 2.4 × (2.5)^1.05
= 2.4 × 2.63
= 6.31 Person-Months

Duration (D) = c × (E)^d
= 2.5 × (6.31)^0.38
= 2.5 × 2.18
= 5.45 Months

Average Staff = E / D
= 6.31 / 5.45
= 1.16 ≈ 1-2 Persons
```
